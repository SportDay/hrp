Ici se trouvent les images des comptes publiques.

Pour rappel.
Un utilisateur qui n'a pas de compte publique aura pour image : default.png
Un utilisateur avec un compte publique se vera attribuer un numero d'image publique (non choisis).

Les premieres images correspondent aux roles existants.
Elles sont au format webp. (exemple : profile_0005.webp)

Les images qui suivent correspondent sont celles des rôles généré proceduralement.
Elles sont au format jpg.  (exemple : profile_0040.jpg)
